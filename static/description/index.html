Módulo de Hojas de Trabajo Diarias para Odoo

Este módulo gestiona el registro diario de actividades en obras y proyectos, con control de firmas, evidencias fotográficas y materiales utilizados.

Características Principales

  • Registro completo de trabajos diarios con firmas digitales
  • Sistema de validación con mínimo 3 fotos por día de trabajo
  • Control de horarios y desplazamientos
  • Gestión de materiales utilizados en cada jornada
  • Flujos de trabajo configurables (borrador, en progreso, finalizado)
  • Generación de reportes en PDF
  • Integración con Partners (clientes) y módulo HR (empleados)

Estructura del Módulo

1. parte.trabajo: Hoja principal de trabajo

2. hoja.trabajo.line: Líneas diarias de trabajo

3. hoja.trabajo.imagen: Imágenes de evidencia

Instalación

  1. Instalar el módulo como cualquier otro módulo de Odoo
  2. Reiniciar el servidor de Odoo
  3. Actualizar la lista de módulos
  4. Instalar el módulo

Configuración

Después de la instalación, el módulo está listo para usar. Se recomienda:

  • Configurar empleados en el módulo HR
  • Tener clientes registrados como Partners
  • Configurar productos/materiales en el inventario

Uso del Módulo

  1. Crear una nueva hoja de trabajo asociada a un cliente
  2. Añadir líneas diarias con horarios, materiales y observaciones
  3. Subir mínimo 3 imágenes por día de trabajo
  4. Registrar firmas del montador y cliente
  5. Cambiar estados según progreso del trabajo
  6. Generar reporte PDF cuando esté finalizado

Validaciones Importantes

  • Campos obligatorios: Cliente, Obra, Fecha
  • Mínimo 3 imágenes por día de trabajo
  • Validación automática al guardar registros incompletos

Notas para Desarrolladores

El módulo utiliza:

  • Campos computados para URLs de firmas
  • Validaciones con @api.constrains
  • Métodos para cambiar estados del workflow
  • Relaciones Many2one, One2many y Many2many
